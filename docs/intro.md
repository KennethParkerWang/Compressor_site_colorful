# 压缩器调研 / Compressor Research Literature Map

本项目是一个科研风格的文献地图展示网站,系统梳理无损压缩算法、PAQ/CMIX 系列、神经通用压缩、图像/高位深/医学/科学数据压缩、Benchmark、可复现资源与研究路线图。

## 主要内容

- **Overview / 项目总览**:研究方向、统计与核心叙事
- **Literature Map / 文献地图**:可缩放、可拖拽的研究方向节点图
- **Core Papers / 核心文献**:核心论文与里程碑式工作
- **Reproducible / 可复现资源**:开源实现、数据集、Benchmark
- **Standards / 标准与基准**:国际/行业标准与基准方法
- **Database / 文献数据库**:可搜索、可筛选的完整文献表

## 数据来源

所有数据来源于 `src/data/` 目录下的数据文件,后续会持续扩充。
